本周焦点

S化妆品新原料备案新规正式施行，高风险功效从10类缩至5类改变原料创新监管框架

7月15日，《化妆品新原料注册备案及资料管理规定》（国家药监局2026年第59号公告）及中检院配套《化妆品新原料注册备案资料技术通则》同步施行，2021年第31号公告同时废止。需注册的"较高风险"功效由防腐、防晒、着色、染发、祛斑美白、防脱发、祛痘、抗皱、去屑、除臭10类缩减为前5类，防脱发、祛痘、抗皱、去屑、除臭5类新原料由注册调整为备案管理。功能依据、检测方法文本、稳定性试验等部分资料调整为企业留档备查；安全性试验可采用国际权威机构收录的替代试验方法；新原料与关联产品可同步申报（国家药监局公告 / 中国经济网 / 人民日报健康客户端）。

同日，国家药监局综合司公开征求《关于〈化妆品安全技术规范（2015年版）〉修订内容实施有关事宜的公告（征求意见稿）》意见，征求意见期至7月21日。征求意见稿明确：涉及禁限用、准用原料目录调整的已上市产品，若仅将相关原料替换为基质性原料或仅降低用量，可通过提交配方变更说明和安全评估完成合规，无需重新注册备案（中国食品药品网）。

影响推演：5类功效新原料由注册调整为备案管理，直接压缩功效型护肤原料的上市周期。替代试验方法接受范围扩大，为国际品牌原料在华注册和国货原料出海降低重复测试成本。管理逻辑由"资料齐备后再申报"转向风险分级与企业主体责任并行。 研发周期缩短不等于安全评价要求降低。原料安全证据、工艺稳定性和产品应用验证的完整度，将决定谁能把制度红利转成实际新品速度。

一、品牌动态

A倩碧一周连发两款重组PDRN产品，雅诗兰黛集团以成分密度加速妆械协同布局

7月13日，倩碧在中国发布CX肌源深修水光系列，为全球首款以重组PDRN为核心成分的护肤产品，原料通过AI数据库从99条DNA序列中筛选，以微生物发酵生产替代传统三文鱼提取（36氪 / 上观新闻）。

7月18日，倩碧官宣《乘风2026》总冠军曾沛慈为品牌新生大使，同步推出全球首款2重组PDRN童颜系列（童颜次抛精华与面膜），搭载2重组PDRN与重组人源化III型胶原蛋白，临床实测4周屏障强韧20%、弹性+19%、眼角细纹-14%（QQ新闻 / 倩碧官方微博）。

影响推演：一周内两次PDRN官宣，雅诗兰黛集团把"首个"和"首款2重组"的占位叙事同时落地。竞争焦点将从成分概念转向递送技术与临床数据的可信度。 医美成分家用化窗口期有限，国货跟进PDRN时需同步建设透皮递送证据链，避免停留在概念层面。

A欧莱雅与联合利华同台WAIC 2026，美妆巨头AI竞争升级为组织基础设施

7月17日，欧莱雅首次参展WAIC 2026，展示3CE GENBA生成式美容顾问、PPD美发造型魔镜、头发数字孪生、CreAltech生成式AI美妆内容实验室等应用（新浪财经 / 果壳）。

7月19日，欧莱雅在WAIC举办"AI向美"论坛，与火山引擎签署战略合作备忘录，并启动美妆科技黑客松。同日，联合利华发布AI赋能包装创新平台"AI for Packaging"，称包装设计周期缩短50%、创意方案产出提升10倍（欧莱雅微博文章 / 新浪财经）。

影响推演：欧莱雅绑定火山引擎意味着AI将嵌入业务逻辑，联合利华把AI延伸至包装设计说明包装已成上市效率核心变量。巨头AI竞争已从单点工具进入组织能力层面的基础设施改造。 国货需区分"效率工具"与"组织能力"两层价值，把参展等同于系统性转型会误判竞争节奏。

A爱美客、敷尔佳、HBN同周深化研发叙事，国货功效竞争进入递送证据阶段

7月16日，爱美客在杭州发布"全层年轻化"战略及三款新品：注射用A型肉毒毒素"解索橙"、射频皮肤治疗仪"嗨塑"、头皮水光"元源出"，覆盖动态表情、真皮脂肪紧塑与毛发三个维度（爱美客官方微博 / 中国经济网）。

7月17日，敷尔佳在PCT峰会公布Cellpolypid® PDRN靶向递送与RE-ANTITILOLO®纳米球胶原两套仿生透皮技术：磷脂包裹PDRN整体渗透率提升50%、到达真皮层提升233%，纳米球胶原促渗效率提升3倍（美通社PR Newswire）。同日，HBN开启"冰美式"眼霜节，主推咖啡因眼霜3.0，以"咖啡发酵滤液"独家新原料绑定熬夜场景（HBN官方微博）。

影响推演：三家国货分别从战略体系、递送技术、原料独家性三个维度加码，说明功效品牌竞争已进入"递送证据+战略闭环"阶段，单纯讲成分浓度的窗口正在收窄。 医用修护品牌向日常抗衰延伸的关键，在于能否将械字号积累的专业信任迁移到妆字号产品。

A兰蔻一周内完成代言人升title与限定唇膏上市，高端品牌把代言与新品节奏压缩在同一周

7月16日，兰蔻官宣张婧仪升任全线品牌大使、张凌赫升任护肤大使，线下大屏分别落址上海西岸梦中心和成都东郊记忆（新浪娱乐 / 新浪娱乐）。

7月18日，兰蔻上市菁纯唇膏「声色」限定系列，灵感源自黑胶唱片声槽纹路，推出4款新色，全球代言人倪妮与张婧仪、张凌赫同步站台（兰蔻官方微博）。

影响推演：兰蔻把代言人升title和限定唇膏上市压缩在同一周，形成"头衔升级制造话题→新品承接流量"的闭环。头衔层级差异也折射出品牌在艺人排他协议下的精细平衡。 高端品牌代言与新品节奏同步加密，要求跟进品牌在代言官宣节点提前备好新品承接链路。

B资生堂与DIOR同期推生物科技底妆护肤新品，高端底妆养肤化赛道竞争加密

7月17日，资生堂SHISEIDO"エッセンス スキンスムース ファンデーション"在日本正式发售，采用"セラムファースト（精华优先）"技术，让美容液先接触肌肤后再形成持妆膜，定位"无噪哑光"妆效（资生堂日本官网 / 资生堂企业新闻稿）。

7月14日至19日，DIOR花秘匿龄限时展在上海西岸中環举办，全新花秘逆转面霜首次面向公众亮相，主打"10维匿龄重塑"，搭载专利复合玫瑰逆转肽™-3（DIOR美妆微博）。

影响推演：资生堂补全哑光养肤分支，DIOR以"逆转细胞年龄"切入再生医学抗老叙事。高端底妆/护肤的生物科技叙事同步加码，品牌需在"持妆"与"养肤"两个维度同时建立可信证据。

B赵露思ROSE AMIGO以自主持股品牌切入颈膜细分赛道，明星美妆走向资本自主

7月11日至12日，赵露思在澳门举办"Stay Romantic"演唱会，个人美妆品牌ROSE AMIGO首次公开亮相，向购票粉丝派发"粉漾盈彩唇膜"限定伴手礼。品牌商标申请人为成都马宁有限企业管理有限公司，赵露思持股90%。首款备案产品为"耀萃焕颜红毯面颈膜"，以烟酰胺、积雪草、可溶性胶原为核心，选择以颈膜而非面膜切入市场，正式产品预计8月上线（美妆网 / 36氪 / 界面新闻）。

影响推演：赵露思从"代言人"升级为自主持股创始人，与多数明星美妆的"贴牌分红"模式有本质区别。选颈膜规避面部红海，策略层面有差异化意识。

B国货彩妆以情绪主题与IP联名驱动上新，国际品牌新品节奏同步加速

7月13日，everbab艾蓓拉发布"假日海风"主题新品预告，延续"轻彩妆+情绪化主题"模式（品牌微博）。

二、市场版图

A化妆品社零6月增12.6%伴随超60家企业注销，行业总量增长中尾部加速出清

7月19日，东方证券研报显示，2026年6月限额以上化妆品类零售额同比增长12.6%，显著跑赢同期社零1.0%的增速。2026年618期间，美妆大盘综合电商平台销售额同比增长9.8%，美容护肤类目成交额达425亿元（东方证券研报）。

7月13日，妆知情报数据显示2026年已有超60家化妆品企业完成注销，结合此前广州前6个月55家企业注销《化妆品生产许可证》的报道，注销企业集中在缺乏差异化的中小代工厂和依赖流量的新锐白牌（妆知情报 / 聚美丽）。

影响推演：6月化妆品社零强劲主要受618集中释放拉动，美妆品类在消费大盘承压下仍显示韧性。但总量增长的同时企业数量在收缩，增长正在向头部集中。

三、渠道生态

ATikTok Shop跻身英国第四大美妆零售商，内容电商在成熟市场进入主流零售序列

7月17日，Public First受TikTok委托发布的报告显示，TikTok Shop已成为英国第四大美妆零售商。2025年英国市场美妆品类在TikTok Shop销售额增长60%，约30%英国年轻人（约180万人）将TikTok视为获取时尚美容建议的主要渠道，2025年英国TikTok Shop美妆主题直播购物节目数量增长90%，活跃Shop创作者数量同比增长超72%（亿邦动力 / 微博）。

影响推演：TikTok Shop在英国美妆零售市场进入前四，说明内容电商在成熟市场已跨越早期尝鲜阶段，成为与Boots、Superdrug等传统零售商并列的主流渠道。 对国货出海欧洲而言，直播和短视频基建提供了绕过传统零售壁垒的直达路径，但合规能力和品牌沉淀仍是长期壁垒。

B央视曝光"古法护肤"违规添加白芷，短视频直播渠道成为三无产品主要流通场景

7月19日，央视《财经调查》曝光一批宣称"传承千年宫廷秘方"的护肤产品实为"三无"产品。所谓"七子白古法护肤产品"违规添加国家化妆品禁用原料白芷，部分消费者使用后出现面部刺痛、红肿过敏。多家直播间以"纯手工现磨、无添加"为卖点销售此类产品，部分厂商将产品备案成"保健外用品"规避化妆品监管（央视网）。

影响推演：短视频和直播渠道成为"古法护肤"三无产品的主要流通场景，白芷等禁用中药材的违规添加问题直接指向内容电商的合规监管压力，也暴露了跨品类备案的监管缝隙。

B纪梵希以会员预约压缩线下体验周期，区域百货闭店与B2B展会反映渠道分化

7月17日至19日，纪梵希美妆在上海百空间光二仓库举办"缎光艺境"限时活动，围绕G方管缎光唇膏设置线下体验，通过会员中心小程序预约入场。7月15日，品牌全球彩妆及香氛代言人赵丽颖出席活动拉开序幕（纪梵希美妆官方微博转载页 / 微博）。

7月18日，OLAY携活力护肤大使宋威龙做客李佳琦直播间，主推水光小白瓶与美白水乳组合，为新任大使官宣后的首次大型达人带货动作（OLAY官方微博）。7月16日，第41届南京美业博览会在南京国际展览中心开幕（7/16-7/18），展会面积超2.5万㎡，500+展商参展，预计专业观众超3万人次（5588.tv）。7月19日，丹尼斯百货政和店（河南洛阳，2012年开业）结束营业后正式闭店（商超网）。

影响推演：高端唇妆的线下投入已转向"内容场景+会员预约"组合，短周期活动把新品试色、品牌叙事和私域留资压缩在一次到店体验中。区域百货闭店与B2B展会高峰并存，说明线下渠道正在分化：大众区域百货收缩，专业B端选品和品牌线下体验场景仍在投入。

四、投融资

A维琪科技申购与半亩花田遭六连问同日出炉，原料企业与流量品牌在资本市场走出分化路径

7月13日，深圳市维琪科技股份有限公司（920176.BJ）在北交所开启申购，发行价22.16元/股，发行市盈率14.99倍。公司拥有22个NMPA备案新原料（其中9个创新新原料），多肽原料市占率6.6%，2023-2025年营收分别为1.65亿、2.48亿、2.75亿元，净利润4229万、7064万、9085万元，客户覆盖宝洁、联合利华、资生堂、雅诗兰黛及丸美、溪木源、HBN等国货品牌（北京商报 / 21世纪经济报道）。

同日，半亩花田母公司山东花物堂在冲刺港股途中收到证监会《境外发行上市备案补充材料要求》，六大问询方向涵盖家族控股超85%表决权、上市前夕突击入股推高估值42亿、股权激励合规性、特殊股东权利、业务资质、全流通股份权属。半亩花田2025年前三季度营收18.95亿元（同比增76.7%），同期营销费用近9亿元，研发仅两三千万（腾讯新闻 / 网易）。

7月19日，深交所官网显示，苏州锦艺新材料科技股份有限公司创业板IPO状态从"已受理"变更为"已问询"，公司拟募集资金约22.19亿元，从事先进无机非金属粉体材料研发生产（QQ新闻 / 新浪财经）。

五、原料与监管

B英国4-MBC与甲醛释放体限制生效，出海英国合规需脱离"欧盟即英国"简化判断

7月15日，英国《Cosmetic Products Regulation (EC) No 1223/2009 (Restriction of Chemical Substances) (Amendment and Transitional Provisions) Regulations 2026》在英格兰、威尔士和苏格兰生效。法规针对4-MBC及甲醛释放体标签规则作出调整，不符合当日新要求的化妆品不得再投放市场，既有库存销售过渡期至2027年1月15日（英国法规原文 / 上海市商务委转载解读）。

影响推演：英国法规已与欧盟形成独立节奏。对计划进入英国的中国防晒及洗护品牌，配方复核不能沿用"欧盟合规即英国合规"的简化判断。历史配方中的4-MBC和甲醛释放体应同步检查英国Responsible Person文件、包装警示语与库存切换计划。
